Data Binding

Data binding allows to define communication between a component and the DOM

Four Ways
  • String Interpolation
  • {{interpolation}}
  • Event binding: Whenever input event is fired, expression/method will be executed.
  • {{eventBinding}}
  • Property binding: The way of binding the value property of element with the variable name.
  • {{propertyBinding}}
  • Two-way data binding: For this we use [(ngModel)]. To use ngModel directive, we need to import FormsModule. 
  • {{twowayBinding}}

Directives and Pipes

Directives

Directives are used to change the appearance, behaviour or layout of DOM element

Categories
  • Component directives: Directives with a template
  • Structural directives: Directives used to change the structure of DOM elements like *ngIf, *ngFor, *ngSwitch
  • Attribute directives: Directives used to change the look and behavior of DOM elements like [ngClass], [ngStyle]
  • Custom directive: We can generate directive using CLI and assign a selector to @Directive
  • {{customDirective}}

Pipes

Pipes are simple way to transform values in an Angular template. It takes integers, strings, arrays, and date as input separated with | to be converted in the format as required.

Two Types
  • Built-in Pipes: Lowercasepipe, Uppercasepipe, Datepipe, Currencypipe, Jsonpipe, Percentpipe, Decimalpipe, Slicepipe
  • Custom Pipes
    • Create a custom pipe using ng g p <pipe-name>
    • Register the custom pipe: CLI will register/add the newly created custom pipe to the @NgModule declarations in app.module.ts
    • Add logic for pipe
      • {{pipe}}
    • Implement the custom pipe
      • In Template: refer to the above code block
      • In Component: Add pipe in the providers array in the metadata and inject in the constructor and invoke the transform() in the pipe.

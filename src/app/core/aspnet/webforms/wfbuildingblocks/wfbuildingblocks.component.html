Building Blocks

Page Life cycle

Phases
  • Initialization
  • Instantiation of the controls on the page
  • Restoration and maintenance of the state
  • Execution of the event handler codes
  • Page rendering
Stages
  • Page request
  • Starting of page life cycle
  • Page Initialization
  • Page load
  • Validation
  • Postback Event handling
  • Pge rendering
  • Unload
Events
  • PreInit: Checks the IsPostBack property
  • Init: Initializes the control property
  • InitComplete: Allows tracking of view state
  • LoadViewState: ALlows laoding view state information into the controls
  • LoadPostData: contents of all input fields are defined with form tag are processed
  • PreLoad: occurs before PostBack
  • Load: load event will raise first for the page and loads for all child controls
  • LoadComplete: loading is completed and the control event handlers are run and page validation occurs
  • PreRender: occurs just before output is rendered
  • PreRenderComplete: Ensures the completion of the PreRender phase
  • SaveStateComplete: state of control on this page is saved
  • UnLoad: raise the unload event for all controls recursively and lastly for page itself

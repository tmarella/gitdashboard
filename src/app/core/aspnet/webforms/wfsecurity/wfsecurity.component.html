Security

Implementing security has following aspects

  • Authentication: Validating based on login credentials
  • {{syntax}}
    • Windows Authentication: Needs to be configured in IIS
      • Four different methods: Anonymous, basic, digest and windows integrated
    • Forms Authentication
      • Configure the Security Settings in the Web.config File
      • Create a Login Page
      • On Submitting the credentials, validate the user credentials by looking in the database using ADO.NET
      • Redirecting based on success/fail authentication
    • Passport Authentication: Uses Microsoft's passport service to authenticate.
  • Authorization: Allowing authenticated users based on roles

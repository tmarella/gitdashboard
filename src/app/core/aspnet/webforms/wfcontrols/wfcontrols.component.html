Controls

Webforms have HTML Server controls and user defined controls

HTML Server Controls

HTML server controls are HTML elements that contain attributes to accessible at server side using runat="server"

Few HTML controls are

  • label, textbox, button
  • listbox, checkbox, dropdownlist
User defined controls

A UserControl is a separate, reusable part of a page. You can put a piece of a page in a UserControl, and then reuse it from a different location

Register User Controls and Custom Controls in Web.config like below

  • Declare them once within the controls section in the web.config file
  • {{registerControl}}
  • the "~" keyword in ASP.NET means "resolve from the application root path"
  • Once you register the controls within the web.config file, you can then just use the controls on any page
  • {{usageRegisterControl}}

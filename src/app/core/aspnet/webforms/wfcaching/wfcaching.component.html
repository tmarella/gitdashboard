Caching

Caching is a technique for storing frequently used data in the memory.

Five Ways
  • Output Caching: Stores the final rendered html pages. This occurs on client side
  • Data Caching: Caching the data from the data source. This occurs on client side
  • Object Caching: Caching the object on the pages such as databound controls. This occurs on server side
  • Class Caching: Caches compiled pages in the assembly. This occurs on server side
  • Configuration Caching: Caches application wide configuration information. This occurs on server side

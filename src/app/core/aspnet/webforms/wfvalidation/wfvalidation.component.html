Validation

Validation controls validate the user input data to ensure that useless, unauthenticated, or contradictory data don't get stored.

Controls
  • RequiredFieldValidator To make sure that the user has entered something
    • {{required}}
  • RangeValidator To make sure that the user input is within a specified range like validating both numbers, strings and dates
    • {{range}}
  • CompareValidator To compare two values, for instance the values of two controls.
    • {{compare}}
  • RegularExpressionValidator To check the validity of any kind of string.
    • {{regex}}
  • CustomValidator Allows writing application specific custom validation routines for both the client side and the server side validation.
    • Client side validation
      • The client side validation is accomplished through the ClientValidationFunction property.
      • {{customclient}}
    • Server side validation
      • The server side validation routine must be called from the control's ServerValidate event handler.
      • {{customserver}}
  • ValidationSummary The ValidationSummary control does not perform any validation but shows a summary of all errors in the page.
    • {{summary}}

Emails

The .NET framework comes with an entire namespace for handling e-mails, the System.Net.Mail namespace.

  • Two classes used are 'MailMessage' class, for the actual e-mail, and the 'SmtpClient' class, for sending the e-mail.
  • Mails are sent through an SMTP server
  • {{email}}
  • To send complicated layouts in the email body, use below
    • {{htmlemail}}

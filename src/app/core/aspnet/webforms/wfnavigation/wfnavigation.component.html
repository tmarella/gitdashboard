Navigation

ASP.NET has built-in navigation controls

Four Ways
  • Sitemap: The menu can be stored in a file normally called web.sitemap, and is stored in the root directory of the web.
    • {{sitemap}}
    • The XML file must contain a siteMap tag surrounding the content
    • The XML file must contain a siteMap tag surrounding the content
    • Each siteMapNode can have several child nodes (web pages)
    • Each siteMapNode has attributes defining page title and URL
  • Dynamic: The <asp:Menu > control displays a standard site navigation menu. 
    • {{dynamic}}
    • The <asp:SiteMapDataSource> control automatically connects to the default sitemap file (web.sitemap).
  • TreeView: The <asp:TreeView >control displays a multi level navigation menu.
    • {{treeview}}
    • The <asp:SiteMapDataSource> control automatically connects to the default sitemap file (web.sitemap).
    • The menu looks like a tree with branches that can be opened or closed with + or - symbol.
    • The <asp:SiteMapDataSource> control automatically connects to the default sitemap file (web.sitemap).
  • SiteMapPath: The SiteMapPath control displays the navigation path to the current page. The path acts as clickable links to previous pages.
    • {{sitemappath}}
    • SiteMapPath control does NOT use a SiteMapDataSource. The SiteMapPath control uses the web.sitemap file by default.

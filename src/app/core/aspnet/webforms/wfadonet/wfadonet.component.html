ADO.NET

ADO.NET provides a bridge between the front end controls and the back end database.

DataSet represents subset of database
DataTable represents tables in the DataSet
DataRow represents row in the DataTable
SqlDataAdapter is an intermediate layer which acts as a bridge between the DataSet and a Database
SqlDataReader Provides a way of reading a forward-only stream of rows
SqlCommand Represents a TSQL statement or stored procedure to execute against a SQL Server database
SqlCommand methods
  • ExecuteNonQuery(): Executes a Transact-SQL statement against the connection and returns the number of rows affected.
  • ExecuteReader(): Returns the collection of rows selected by the Query.Return type is DataReader and must be assigned to another object DataReader.
  • ExecuteScalar(): Executes the query, and returns the first column of the first row. Return type is object and must bea assigned to a variable of required type.
  • ExecuteXmlReader(): Sends the CommandText to the Connection and builds an XmlReader object.
SqlCommand properties
  • CommandText: Gets or Sets the table name or stored procedure to execute
  • CommandType: Gets or sets a value indicating how the CommandText property is to be interpreted.
  • Connection: Gets or sets the SqlConnection
  • Parameters: Gets the SqlParameterCollection

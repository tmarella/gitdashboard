Filters

Web API includes filters to add extra logic before or after action method executes like logging, exception handling, performance measurement, authentication and authorization.

Few interfaces and classes that can be used to create Web API filters.

Filter Type Description
Simple Filter Defines the methods that are used in a filter
Action Filter Used to add extra logic before or after action methods execute.
Authentication Filter Used to force users or clients to be authenticated before action methods execute.
Authorization Filter Used to restrict access to action methods to specific users or groups.
Exception Filter Used to handle all unhandled exception in Web API
Override Filter Used to customize the behaviour of other filter for individual action method.

Api Routing

Web API routing is similar to MVC routing

Convention based Routing
  • Url Pattern: {api}/{controller}/{id}
  • The config.Routes is a route table or route collection of type HttpRouteCollection
  • The "DefaultApi" route is added in the route table using MapHttpRoute() extension method
  • {{addRoute}}
  • Web API also supports routing same as ASP.NET MVC by including action method name in the URL.
  • Multiple routes can be added like below
  • {{addMultipleRoutes}}
Attribute based Routing
  • Attribute routing uses [Route()] attribute to define routes.
  • Attribute routing is enabled in WebApiConfig.cs by calling config.MapHttpAttributeRoutes() method.
  • {{attributeRoute}}
Parameter Binding
  • Default Web API gets the value of a primitive parameter from the query string and complex type parameter from the request body but we can change it using FromUri and FromBody
  • FromUri: attribute to get the value of complex data type from the query string
  • FromBody: attribute to get the value of primitive data type from the request body

Building Blocks

The standard SQL commands to interact with relational databases are CREATE, SELECT, INSERT, UPDATE, DELETE and DROP.

Groups

DDL- Data Defination Language

Create Creates new table, function, view, SP etc..
Alter Modifies an exsisting DB object
Drop Deletes an entire table, views etc..

DML- Data Manipulation Language

Select Retrieves certain records from one or more table
Insert Creates records
Update Modifies records
Delete Deletes records

DCL- Data Control Language

Grant Gives a privilage to user
Revoke Remove previlage from user.

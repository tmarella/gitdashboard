Joins

Join clause is used to combine records from two or more tables.

Four Types
  • {{joinsSyntax}}
    • Inner Join: Returns rows when there is a match in both tables.
    • Left Join: Returns all records from left table, and the matched records from right table.
    • Right Join: Returns all records from right table, and the matched records from left table.
    • Full Join: Returns all records when there is match in either left or right table.

    Unions

    The Union Clause is used to combine two separate select statements.

    The fields used in both the select statements must be in same order and same data type.

    The Union clause produces distinct values in the result set

    To fetch the duplicate values, UNION ALL must be used

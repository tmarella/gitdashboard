Stored Procedures

SQL Server stored procedures are used to group one or more Transact-SQL statements into logical units that will be stored for later use and can be used many times

'Create Procedure' is used for wrapping the sql statements

  • {{spSyntax}}
    • Parameters: SP allows single or multiple parameters. IN(receives the values from calling program.), OUT(sends the values to the calling program.), IN OUT(receives value from as well as sends the values to the calling program).
    • Variables: We can declare a variable inside SP using DECLARE
    • The BEGIN...END statement is used to define a statement block. A statement block consists of a set of SQL statements that execute together. A statement block is also known as a batch.
    • We can manage the control flow using IF ELSE, WHILE, BREAK, CONTINUE
    • CURSOR is used to process a data set on a row by row basis
    • Exceptions can be handled using TRY CATCH, RAISEERROR, THROW

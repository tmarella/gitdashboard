Functions

Sql has many built-in functions for doing operations on data.

Aggreegate Functions
Aggreegate Functions are used to do operations from the values of the column and a single value is returned.
AVG() Returns average value after calculating from values in a numeric column
COUNT() Returns count of number of rows returned in a SELECT statement
MAX() Returns max value of the selected column
MIN() Returns min value of the selected column
FIRST() Returns first value of the selected column
LAST() Returns last value of the selected column
SUM() Returns the sum of all the values of the selected column
Scalar Functions
These functions are based on user input, these too returns single value.
UCASE() Converts the value of the field to upper case
LCASE()() Converts the value of the field to lower case
Len Returns the length of the value in a text field
ROUND() Used to round a numeric field to the number of decimals specified
FORMAT() Used to format how a field is to be displayed.
NOW() Returns the current system date and time
More Built-in Functions
Concat Adds two or more strings together
Reverse Reverses the string and returns
Trim Removes leading and trailing spaces
Cast Converts a value in to specified datatype
Coalesce Returns first non-null value in a list

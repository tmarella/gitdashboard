Abstraction

Abstraction is a principle used to hide the implementation details and display only essential features of the object.

  • We can acheive abstraction by using 'abstract' keyword for a class/method.
  • Abstract methods will not have implementation.
  • We need to write the implementation in a method with same name in derived class.
Advantages
  • Reduces complexity.
  • Avoids code duplication.
  • Increases reusability and security of the application.

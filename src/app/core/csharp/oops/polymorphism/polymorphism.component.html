Polymorphism

Polymorphism provides ability to a class to have multiple implementations with same name.

Three Types
  • Compile Time Polymorphism or static binding or early binding.
    • Implemented using method overloading.
    • we can perform different tasks with the same method name by passing different parameters.
  • Run Time Polymorphism or dynamic binding or late binding.
    • Implemented using method overriding.
    • number of parameters and parameter type must be same.
    • Declare a base class method as virtual and derived class method as override
    • The virtual keyword is used to modify a method, property, indexer, or event declared in the base class and allow it to be overridden in the derived class.
    • The override keyword is used to extend or modify a virtual/abstract method, property, indexer, or event of base class into derived class.
  • Method Hiding
    • Method Hiding is when we use the 'new' keyword to hide a base class member.
    • When parent class doesn't give permission to implement the method in derived class, 'new' keyword is used to re-implement the base class method in derived class.

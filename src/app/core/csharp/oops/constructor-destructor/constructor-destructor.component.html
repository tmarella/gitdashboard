Constructor and Destructor

Constructor

Constructor is a special method that is used to initialize objects.

  • Constructor name must match the class name, and it cannot have a return type.
  • Constructor is called when the class is instantiated.
  • Constructor can also take parameters, which is used to initialize fields and saves time.
  • Note: Constructor cannot be static, abstract, final and synchronized.
  • Note: Constructor do not have a return type.
Advantages
  • Proper intialization of an object is ensured.
  • Eliminates placing the default values.

Destructor

Destructor is a method used to destroy instance of that class.

  • There cannot be more than one destructor in a class
  • Destructor is distinguished by '~' prefixed to its name
  • Destructor cannot be overloaded or inherited and cannot accept any parameters or modifiers.

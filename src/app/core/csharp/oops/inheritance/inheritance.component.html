Inheritance

Inheritance is a concept used to inherit the properties from one class (base) to another (child) class.

  • We can implement inheritance using access modifiers.
  • We can reuse, extend and modify the behavior of other class members.
Advantages
  • Reduces code redundancy.
  • Improves code readability and easy to manage.
Disadvantages
  • Parent and child classes are tightly coupled and cannot be used independently.

Multiple Inheritance

In Multiple inheritance, one class can have more than one parent class and can inherit features from all its parent classes

  • We use interfaces to achieve multiple class inheritance

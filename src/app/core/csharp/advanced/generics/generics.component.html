Generics

Generics allows to delay the specification of the data type of programming elements in a class or a method until it is actually used in the program. In other words, generics allow you to write a class or method that can work with any data type.

Syntax
  • To declare class as Generic, [class_name]<T>
  • To instantiate generic class, [class_name]<any type>
  • To declare method as Generic, [method_name]<T>
  • To instantiate generic method, [method_name]<any type>
  • To declare parameters as Generic, [method_name](T param1, T param2)
  • To instantiate method in generic class, [class_name]<any type>.[method_name](any type param1, any type param2)
  • To implement generic interface: class Stack<T> where T : System.IComparable<T>, IEnumerable<T>
Advantages
  • Maximize code reuse, type safety, and performance.
  • Generics will eliminate the boxing and unboxing operations.

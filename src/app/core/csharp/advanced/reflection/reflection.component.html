Reflections

Reflection is used to obtain type information at runtime using classes that give access to the metadata of a running program.

Common usage
  • log4net.LogManager.GetLogger(System.Reflection.MethodBase.GetCurrentMethod().DeclaringType);
  • typeof(T)
Advantages
  • Can view metadata
  • Check the type of an object at runtime like typeof(), GetCurrentMethod()
  • Inspect the Attributes of an object at runtime to change the behavior of a method
Disadvantages
  • Overhead at runtime.
  • Performance can be hampered a lot

Type casting or Type conversion

Type casting is a mechanism to convert one data type value to another one.

Type conversion is possible if both the data types are compatible to each other; otherwise you will get an InvalidCastException.

Three Types
  • Implicit casting
    • Implicit conversion is being done automatically by the compiler and no data will be lost
    • Includes conversion of a smaller data type to a larger data types
    • Includes conversion of derived classes to base class
    • This is a safe type conversion.
    • {{implicit}}
  • Explicit casting
    • Explicit conversion is being done by using a cast operator.
    • Includes conversion of larger data type to smaller data type
    • Includes conversion of base class to derived classes.
    • In this conversion information might be lost or conversion might not be succeed for some reasons.
    • This is an un-safe type conversion.
    • {{explicit}}
  • User-defined casting.
    • User-defined conversion is performed by using special methods that you can define to enable explicit and implicit conversions.
    • Includes conversion of class to struct or basic data type and struct to class or basic data type.
    • All conversions methods must be declared as static.
  • Upcasting: Implicit conversion of derived classes to base class
  • Downcasting: Explicit conversion of base class to derived classes

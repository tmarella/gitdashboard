Delegates

Delegates are used when we want to pass a method/function as a parameter using 'delegate' keyword.

Scenario
  • I want you to write a method in the Employee class, which can be used to promote employees.
  • The logic based on which the employee gets promoted should not be hard coded because we may promote employees based on their experience and at times we may promote them based on their salary or may be some other condition.
  • By implementing delegates here, anyone who uses the Employee class will have the flexibility of determining the logic on how they want to promote their employees
Advantages
  • Delegates are type safe.
  • Delegate instances attach or detach a method at run time making it more dynamic and flexible to use
  • Delegates can invoke more than one method using the Multicast feature

Exception Handling

Exception Handling is implemented to prevent crashing of program.

Keywords
  • Try: block that contains actual code/logic
  • Catch: block that catches exceptions like IndexOutOfRangeException, NullReferenceException, InvalidCastException and handles it
  • Finally: block that is used to execute logic, whether an exception is thrown or not thrown.
  • Throw: keyword used to throw an exception explicitly
Advantages
  • Separating Error Handling Code from Regular Code.

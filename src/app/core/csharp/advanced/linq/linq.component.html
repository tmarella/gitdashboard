Linq

Language Integrated Query offers easy data access from in-memory objects, databases, XML documents, and many more

Syntax
  • Lambda syntax → words.Where( w â‡’ w.length > 10);
  • Query syntax → from w in words where w.length > 10;
Advantages
  • Compile time syntax checking.
  • LINQ offers IntelliSense which means writing more accurate queries easily.
  • Easy debugging
  • Offers facility of joining several data sources
  • Allows querying collections in the native language like VB or C#
Five Types
  • LINQ to Objects
    • Allows retrieval of data from collections easily
  • LINQ to XML(XLINQ)
    • Allows working with in-memory XML
  • LINQ to Dataset
    • Allows querying the dataset
  • LINQ to SQL
    • Allows querying the database context files(.dbml) and access the sql server database
  • LINQ to Entities
    • Allows querying entity framework objects

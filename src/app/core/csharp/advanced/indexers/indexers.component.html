Indexers

Indexers allow instances of a class or struct to be indexed just like arrays using 'this' keyword. We will use get and set accessors for defining an indexer.

Advantages
  • Used for overloading [] operator.
  • Supports overloading for any user-defined array.
  • Syntax is simple and easy to use

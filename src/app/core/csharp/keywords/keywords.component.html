Keywords

Keywords are reserved words that have special meaning for compiler.

async is used to specify that a method is asynchronous
await suspends evaluation of the async method
override is used to extend or modify the abstract implementation of a base class method
sealed modifier prevents other classes from inheriting from the class
static modifier makes an class non-instantiable. static when applied to variable, method, they can be accessed without creating an object of the class.
event is a notification sent by an object
enum is used to declare a list of named integer constants like traffic lights, weekdays, months

Grid
  • The Essence of Layouts with Bootstrap’s Rows & Columns
  • Grid: is the combination of the Container, Rows & Columns that make responsive layouts possible
  • Container: The root (top-level) element of the Bootstrap grid is the Container and it is important to control width of the layout. Bootstrap provides two types of containers
    • fixed-width container to center your layout in the middle. Usage: div class="container"
    • full-width container for a layout the spans the entire width. Usage: div class="container-fluid"
  • Rows & Columns: Directly inside the containers, we have rows and columns. Rows serve only one purpose and that is to contain columns. Rows are wrappers for columns
  • There are 12 column units that span horizontally (in a row) across the width of the container. Width of each col would be
    • col-1: 8.33333%
    • col-2: 16%
    • col-3: 25%
    • col-4: 33.33333%
    • col-5: 41.666667%
